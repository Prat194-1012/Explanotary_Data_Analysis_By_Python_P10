Diwali Sales Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on a Diwali sales dataset to uncover customer purchasing patterns and business insights. Using Python data analysis libraries, the dataset is cleaned, transformed, and visualized to understand how different demographic and behavioral factors influence sales during the festive season.

The analysis focuses on identifying key customer segments, purchasing power, and regional sales trends to help businesses make data-driven marketing and inventory decisions.

📊 Objectives

The main objectives of this project are:

Perform data cleaning and preprocessing

Analyze customer demographics and buying behavior

Identify high-value customer segments

Understand sales distribution across different states

Visualize trends using data visualization techniques

🛠️ Technologies Used

Python

Pandas – Data manipulation and analysis

NumPy – Numerical operations

Matplotlib – Data visualization

Seaborn – Statistical data visualization

Jupyter Notebook

📂 Dataset

The dataset contains transactional data related to Diwali festival sales, including:

Customer Gender

Age Group

Marital Status

State

Occupation

Product Categories

Number of Orders

Purchase Amount

🔎 Data Analysis Steps
1️⃣ Data Cleaning

Removed unnecessary columns

Checked and handled missing values

Converted data types for accurate analysis

Performed dataset inspection using:

info()

describe()

isnull()

2️⃣ Exploratory Data Analysis

Several factors were analyzed:

👩 Gender Analysis

Compared number of buyers by gender

Analyzed total purchase amount by gender

Insight:
Female customers contribute more to total purchases than male customers.

🎂 Age Group Analysis

Studied purchasing patterns across different age groups.

Insight:
Customers in the 26–35 age group contribute the most to sales.

🌍 State-wise Sales Analysis

Identified top states by orders

Compared total revenue generated by each state

Insight:
Top contributing states include:

Uttar Pradesh

Karnataka

Maharashtra

💍 Marital Status Analysis

Compared purchase behavior of married vs unmarried customers

Insight:
Married women show the highest purchasing power.

💼 Occupation Analysis

Analyzed which professions contribute most to purchases.

This helps businesses identify target customer segments for marketing campaigns.

📈 Key Insights

Women customers dominate purchasing behavior

Age group 26–35 is the most active buyer segment

Married women are a major contributor to festive sales

Certain states drive the majority of orders and revenue

Customer occupation also influences purchasing patterns
